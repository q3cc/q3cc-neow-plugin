# Changelog

## v0.0.10 - 2026-04-04

### 功能调整
- 将签到指令调整为顶级指令 `/sign`
- 新增签到别名：`/签到`、`/qd`、`/checkin`
- `/24g` 菜单中的签到入口同步改为 `/sign`

### 签到系统优化
- 签到提示改为更自然的分段式展示
- 新增“今天第 N 位签到”提示
- 新增“累计签到 N 次”统计
- 签到奖励文案统一改为 `Star币` 与 `大喵喵`
- 去掉“签到成功喵~”和“当前状态:”等冗余提示
- 签到状态面板调整为缩进展示，更便于阅读

### 随机提示词
- 新增 `resources/sign-prompts.json`
- 签到提示改为从 JSON 文件读取
- 内置 100 条独立签到提示，便于后续直接扩展和替换

### 24 点游戏修复
- 修复 24 点对局状态在多次消息处理中可能丢失的问题
- 将对局状态迁移到共享模块存储，避免 `/24g start` 后 `/24g answer ...` 取不到当局数据
- 普通模式去掉“也可以直接发送 y / n”提示，减少误触
- 普通模式不再兼容直接发送 `y` / `n`，统一要求使用 `/24g answer y|n`

### 结构整理
- 保持主插件位于 `apps/neow.js`
- 保持用户数据与签到逻辑位于 `utils/user-data.js`
- 保持 24 点逻辑位于 `utils/game24.js`

### 验证
- 已通过：
  - `node --check index.js`
  - `node --check apps/neow.js`
  - `node --check utils/user-data.js`
  - `node --check utils/game24.js`
