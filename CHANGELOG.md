# Changelog

## Unreleased

### 24 点玩法调整
- 修复 `/24g difficulty <0-3>` 会误读 `24g` 中数字、导致难度设置失败的问题
- `/24g start` 现在禁止同一用户在同一群内重复开启第二局
- 极限模式去掉“也可以直接发送算式或 no”提示，统一要求使用 `/24g answer ...`
- `/24g` 菜单移除 `/sign - 每日签到` 入口，避免与游戏菜单混杂

### 签到与文案调整
- 签到奖励范围调整为 `1-50` 枚 Star 币、`1-30` 点好感度
- 签到结果不再附带状态面板，仅保留签到顺位、累计签到次数与奖励结果
- 统一将相关文案中的 `Star币` 调整为 `Star 币`

### 数据持久化
- 新增 `data/q3cc-neow-plugin/users.json` 用户数据持久化
- 新增 `data/q3cc-neow-plugin/sign-stats.json` 每日签到统计持久化
- 用户的 Star 币、好感度、体力、难度、签到次数、签到状态、临时管理员状态、封禁状态都会在重启后保留
- 新增 `.gitignore`，忽略运行期生成的 `data/` 目录
- 修复持久化数据加载后 QQ 号键类型不一致，导致重启后个人信息与签到状态失效的问题

### 互动与经济系统
- 新增 `/transfer <QQ> <数量>`，支持用户之间转赠 Star 币
- 新增 `/poke` 与 `/戳` 指令
- 新增 `resources/poke-actions.json`，内置 50 条随机互动动作
- `/poke` 与 `/戳` 现在会从 JSON 中随机读取动作回复

### 管理员系统
- 新增 `/su`，用于切换临时管理员身份
- 主人可直接通过 `/su` 切换为管理员
- 非主人执行 `/su` 时，会在日志输出 16 位中英文验证码
- 新增 `/su <验证码>` 验证流程
- 管理员验证码有效期调整为 5 分钟
- 验证成功后临时管理员权限持续 3 分钟，超时自动失效
- 新增 `/demote`，可主动撤销自身临时管理员身份
- `/nhelp` 现在会在管理员状态下显示管理员专用菜单

### 管理员指令
- 新增 `/setcoin <QQ> <数量>`，可直接设置用户 Star 币
- 新增 `/givecoin <QQ> <数量>`，可直接增加用户 Star 币
- 新增 `/rmcoin <QQ> <数量>`，可直接扣除用户 Star 币
- 新增 `/ban <QQ>`，可永久封禁用户
- 新增 `/tempban <QQ> <分钟>`，可临时封禁用户
- 新增 `/unban <QQ>`，可解除用户封禁
- 被封禁用户将无法使用插件内主要指令
- 临时封禁到期后会自动解除
- 管理员加币指令名称由 `/give` 调整为 `/givecoin`

### 验证
- 已通过：
  - `node --check apps/neow.js`
  - `node --check utils/user-data.js`

## v0.0.10 - 2026-04-04

### 功能调整
- 将签到指令调整为顶级指令 `/sign`
- 新增签到别名：`/签到`、`/qd`、`/checkin`
- `/24g` 菜单中的签到入口同步改为 `/sign`

### 签到系统优化
- 签到提示改为更自然的分段式展示
- 新增“今天第 N 位签到”提示
- 新增“累计签到 N 次”统计
- 签到奖励文案统一改为 `Star币` 与 `大喵喵`
- 去掉“签到成功喵~”和“当前状态:”等冗余提示
- 签到状态面板调整为缩进展示，更便于阅读

### 随机提示词
- 新增 `resources/sign-prompts.json`
- 签到提示改为从 JSON 文件读取
- 内置 100 条独立签到提示，便于后续直接扩展和替换

### 24 点游戏修复
- 修复 24 点对局状态在多次消息处理中可能丢失的问题
- 将对局状态迁移到共享模块存储，避免 `/24g start` 后 `/24g answer ...` 取不到当局数据
- 普通模式去掉“也可以直接发送 y / n”提示，减少误触
- 普通模式不再兼容直接发送 `y` / `n`，统一要求使用 `/24g answer y|n`

### 结构整理
- 保持主插件位于 `apps/neow.js`
- 保持用户数据与签到逻辑位于 `utils/user-data.js`
- 保持 24 点逻辑位于 `utils/game24.js`

### 验证
- 已通过：
  - `node --check index.js`
  - `node --check apps/neow.js`
  - `node --check utils/user-data.js`
  - `node --check utils/game24.js`
